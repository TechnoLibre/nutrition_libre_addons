Calculateur d'IMC et de Percentiles pour Enfants

Ce module ajoute un snippet interactif à votre site web, permettant aux utilisateurs de calculer facilement l'Indice de Masse Corporelle (IMC) et les percentiles pour les enfants. Il s'adresse à deux groupes d'âge : les bébés (0-24 mois) et les enfants (24-96 mois). En plus de déterminer l'IMC, il fournit des informations sur les percentiles, aidant ainsi à évaluer le développement physique des enfants en comparant leurs mesures avec les normes de croissance.

Les utilisateurs peuvent entrer les informations suivantes : âge, taille, poids, et choisir les unités de mesure. Le calculateur convertit automatiquement les unités si nécessaire, calcule l'IMC et affiche le résultat avec une interprétation basée sur les normes de croissance actuelles. Des graphiques illustrant les percentiles sont également fournis pour une compréhension visuelle.

Ce snippet est conçu pour être facile à intégrer dans votre site web, offrant une fonctionnalité pratique pour les parents, les pédiatres, et les professionnels de la santé.
